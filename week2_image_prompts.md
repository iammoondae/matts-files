# Google Nano Banana Image Prompts - Grade 3 Study Guide (Week 2)

This document contains a comprehensive set of image-generating prompts for **Week 2** across all six subjects of Grade 3 (Math, Science, English, Filipino, Makabansa, GMRC). There are 5 images per subject, making a total of 30 images.

These prompts follow the **Presentation Image Prompt Generation Framework** (educational value, age appropriateness for Grade 3, visual clarity, consistency, and negative prompting).

---

## Style Guide (Consistent Across All Prompts)
To ensure the study guide feels professional, cohesive, and appealing to Grade 3 students, all prompts request a **consistent visual style**:
* **Aesthetic**: Friendly, clean, 2D vector educational illustration style with clear outlines and soft shadows.
* **Palette**: Clean, harmonious, child-friendly color palettes tailored to the subject theme.
* **Target Audience**: 8-year-old child level. Simple compositions with high clarity and clear visual hierarchy.
* **Important Constraint**: **No text, numbers, letters, labels, or watermarks** inside the generated images.
* **Style-independence clause**: Each prompt includes a distinct stylistic variant descriptor to prevent style bleeding or identical compositions across slides of the same week.
